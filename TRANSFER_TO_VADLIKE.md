# Transfer to `vadlike/vadlike`

Use this checklist to move the final profile setup.

1. Open repository `https://github.com/vadlike/vadlike`.
2. Replace `README.md` with this file from current repo:
   - `README.md`
3. Add workflow file:
   - `.github/workflows/snake.yml`
4. Commit and push to `main`.
5. Open `Actions` tab in `vadlike/vadlike` and run workflow `Generate Snake` once manually.
6. Verify these render on profile:
   - Typing banner
   - Repo cards
   - Stats / streak / trophies / graph
   - Snake animation

If snake does not appear, wait 1-2 minutes after the workflow finishes successfully.
